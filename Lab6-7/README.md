
# Алгоритм решения задачи

## Условие задачи

Дано 2 списка с равным количеством чисел. Вычислите попарную сумму элементов из этих списков (первый элемент из первого списка, первый из второго; второй элемент из первого списка, второй из второго; ...).

## Ввод и вывод

* ввод - числа.
* вывод - попарные суммы.

## Алгоритм

1. Создать 2 списка с равным количеством чисел:

1.1 Объявить переменные списков.
1.2 Вписать числа.

2. Задать структуру хранения результата:

2.1 Создать новый список для хравнения полученных значений.
2.2 Передать в него значение длины одного из списков (так как будет совпадать и не должно превышать).

3. Сложить элементы 2-х массивов:

3.1 Проверить совпадают ли длины списков:

    * Сравнить равна ли длина первого списка длине второго и длине списка результатов.
    * Выдать сообщение в случае несовпадения.
    * Если ошибки нет, то перейти к следующему шагу.

3.2 Пройтись по каждому элементу обоих списков:

    * Создать ячейку памяти для текущего индекса.
    * Вписать в ячейку текущего индекса 0.
    * Проверить, если индекс < длины списка результатов - перейти к выполнению тела цикла, иначе алгоритм завершается.
    * Прибавить элемент по текущему индексу из первого списка к элементу их второго списка.
    * Увеличить индекс на 1.
    * Вернуться на шаг 3.

4. Вывести суммы из списка хранения:

4.1 Пройтись по каждому элементу списка с результатами:

    * Создать ячейку памяти для текущего индекса.
    * Вписать в ячейку текущего индекса 0.
    * Проверить, если индекс < длины списка результатов, перейти к выполнению тела цикла, иначе алгоритм завершается.
    * Вывести элемент по текущему индексу.
    * Увеличить индекс на 1.
    * Вернуться на шаг 3.
